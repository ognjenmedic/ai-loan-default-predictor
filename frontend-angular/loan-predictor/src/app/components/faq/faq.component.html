About This Project

This demo app showcases a deployed Machine Learning (ML) pipeline for loan default prediction. Below, you’ll find details on how to use it, explore the code, and understand its key features.

Frequently Asked Questions

  1. What is this project?
    This is a machine learning demo app built using Python (Flask), Angular, and LightGBM. It predicts loan default risk based on user inputs, using a model trained on the Home Credit Default Risk Kaggle dataset. It is not affiliated with or endorsed by Home Credit Group. The project showcases my expertise in data preprocessing, feature engineering, model training, and app deployment.
  2. How does this app work?
    - Enter loan-related details or click "Generate Test Data" to auto-fill fields.
    - Click "Predict Default" to get a probability score of default risk.
    - The backend Flask API processes the inputs, applies feature engineering, runs the trained LightGBM model, and returns a prediction.
  3. Where can I see the code and notebooks?
    You can find the full project code and notebooks here:
    • 📂 Kaggle (Notebooks) – Includes data processing, feature engineering, and model training notebooks.
    • 📂 GitHub (App Code) – Includes the Flask API, Angular frontend, and ML pipeline.

    If you'd like to see the notebooks on GitHub instead of Kaggle, let me know, and I’ll provide access.

  4. What does this project demonstrate?
    This project highlights end-to-end machine learning deployment, including:
    • Data cleaning & feature engineering (7 datasets merged)
    • Model training (LightGBM with hyperparameter tuning)
    • Full-stack app development (Angular + Flask)
    • Live ML inference via API (Flask + Docker deployment)
    • Automated test data generation (based on dataset statistics)
  5. Is this an official Home Credit tool?
    No. This project is not affiliated with Home Credit Group, nor is it endorsed by them. It is purely a personal portfolio project to demonstrate my skills.
