Frequently Asked Questions

  1. What is this project?
    This is a machine learning demo app built using Python (Flask), Angular, and LightGBM. It predicts loan default risk based on user inputs, using a model trained on fully synthetic data generated by custom scripts.

    While the project structure was originally inspired by open-source credit risk challenges, it does not use any real-world or proprietary datasets. It is intended solely for educational and portfolio demonstration purposes.
  2. How does this app work?
    - Enter loan-related details or click "Generate Test Data" to auto-fill fields.
    - Click "Predict Default" to get a probability score of default risk.
    - The backend Flask API receives the input, uses a pre-trained LightGBM model (trained on synthetic data), and returns a prediction.
